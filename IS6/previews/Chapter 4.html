Notes:

  • If log_scale = True, default log is base 10.
  • log_scale can take values that are bool or number, or pair of bools or numbers. A pair of values will set each axis independently.

Notes:

  • sns.boxplot does not have a log_scale parameter, so if we want to re-express, we would need to call plt.yscale("log") instead. If this helps, think of it as making the box plot first then re-rexpress instead of plotting the re-expressed box plot from the start.
